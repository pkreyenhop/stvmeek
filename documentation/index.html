Voting Systems Toolbox

Voting Systems Toolbox is designed to make implementations of voting systems freely available in Java. It will initially have software for Single Tansferrable Vote (Meek Algorithm), but others may be included later.

It is released under the GNU Public License, and requires Java 1.1 or later to run.

It is being developed by Roy Ward, using Sourceforge for all project maintainance.
